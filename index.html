Google MCP

A high-performance, serverless backend for Google Workspace integration via Model Context Protocol.

Setup Tools Source

Quick Setup

Step 01

Deploy Apps Script

Create a new project at script.google.com. Paste Code.gs and appsscript.json. Set MCP_API_KEY in project settings.

Step 02

Deploy as Web App

Deploy > New Deployment > Web App. Set "Execute as: Me" and "Who has access: Anyone". Copy your deployment URL.

Step 03

Connect Client

Update your mcp_config.json with the following configuration snippet.

"google-mcp": { "command": "npx", "args": [ "-y", "mcp-remote@latest", "https://script.google.com/macros/s/.../exec?key=YOUR_KEY" ] }

Capabilities

Communications

Full Gmail integration. Read, search, send, and reply to threads directly from the LLM.

Storage

Google Drive management. List, search, create, and modify files or directories.

Scheduling

Calendar and Tasks. Manage events and TODOs with natural language processing.

Productivity

Sheets and Docs. Direct data entry and document generation from AI outputs.
