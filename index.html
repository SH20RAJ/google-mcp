Installation Guide

Get up and running in minutes.

Web IDE (Cloud)
Using mcp-remote
Local (clasp)

1 Create Project

Go to script.google.com and create a new project.

2 Paste Code

Copy Code.gs and appsscript.json from the GitHub repository and paste them into the editor.

3 Set API Key

In Project Settings, add a Script Property MCP_API_KEY with your secure secret key.

4 Deploy

Click Deploy > New Deployment > Web App. Set access to "Anyone" and copy your Deployment URL.

Add this to your mcp_config.json to connect your local LLM client to your Google backend.

"google-mcp": {
  "command": "npx",
  "args": [
    "-y",
    "mcp-remote@latest",
    "https://script.google.com/macros/s/.../exec?key=YOUR_KEY"
  ],
  "disabledTools": []
}
